About This Game

Crimson Recoil is a fast-paced FPS game set in a dark, intense atmosphere. Players take on the role of the last soldier in a world ravaged by destruction, battling hordes of mysterious enemies with futuristic weaponry. With realistic recoil mechanics and dynamic level progression, Crimson Recoil challenges your reflexes, strategy, and precision with every shot. This game is a reference from YouTube.

Solo Developer

I built Crimson Recoil completely on my own — from concept, level design, to C# scripting and gameplay systems. This solo project demonstrates my ability to handle a full development cycle using Unity.
